# üìù Note App

A full-stack **MERN (MongoDB, Express, React, Node.js)** note-taking web
application. Users can securely create, edit, and manage notes in real-time.
**Firebase Authentication** is used for secure sign-in/sign-up and session
management. The app features a modern, responsive UI for a seamless user
experience.

![Note App Screenshot](https://note-v1.vercel.app/note.png)

## ‚ö° Features

- **User Authentication:** Secure login and registration using Firebase Auth
- **Real-Time Note Management:** Add, edit, delete, and view notes instantly
- **RESTful API:** Node.js & Express backend with MongoDB for CRUD operations
- **Responsive UI:** Built with React, Tailwind CSS, and DaisyUI
- **Full-Stack Integration:** Seamless connection between frontend and backend

---

## Tech Stack

- **Frontend:** React, Vite, Tailwind CSS, DaisyUI
- **Backend:** Node.js, Express.js, MongoDB
- **Authentication:** Firebase Auth
- **Deployment:** Vercel
